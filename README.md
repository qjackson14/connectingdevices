<h2>Step-by-Step</h2>

- Connect PC1 and PC2 to SW3 and Sw4 using straight-through cables

- Connect PC3 and SRV1 to SW7 and SW8 using straight-through cables

- Connect SW3 and SW4 to SW1 and SW2 using crossover cables

- Connect SW7 and SW8 to SW5 and SW6 using crossover cables

- Connect SW1 and SW2 to R2 using straight-through cables

- Connect SW5 and SW6 to R4 using straigh-through cables

- Connect R2 to R1 with crossover cable

- Connect R1 to R3 using single mode fiber optic cable

- Connect R4 to R3 using multi mode fiber optic cable
